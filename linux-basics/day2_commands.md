# Day 2 - Linux Basic Commands 📂

##  Objective
Learn basic navigation and file handling commands in Linux.

---

##  Commands I Learned

### 1. pwd (Print Working Directory)
Shows current folder path.

Example:
pwd

---

### 2. ls (List)
Shows files and folders.

Example:
ls

---

### 3. cd (Change Directory)
Move between folders.

Examples:
cd foldername
cd ..
cd ~

---

### 4. mkdir (Make Directory)
Create a new folder.

Example:
mkdir myfolder

---

### 5. touch
Create a new file.

Example:
touch file1.txt

---
